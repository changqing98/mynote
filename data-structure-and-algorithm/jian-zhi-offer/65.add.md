# 面试题65. 不用加减乘除做加法

## Problem

写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“\*”、“/” 四则运算符号。

**示例:**

```text
输入: a = 1, b = 1
输出: 2
```

**提示：**

* `a`, `b` 均可能是负数或 0
* 结果不会溢出 32 位整数

## Solution

计算机逻辑运算公式

```java
public int add(int a, int b) {
    while(b != 0){
        int tmp = a ^ b;
        b = (a & b) << 1;
        a = tmp;
    }
    return a;
}
```
