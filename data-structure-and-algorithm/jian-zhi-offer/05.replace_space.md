# 面试题05. 替换空格

## Problem

请实现一个函数，把字符串 s 中的每个空格替换成"%20"

示例 1：

输入：s = "We are happy." 输出："We%20are%20happy."

限制：

0 &lt;= s 的长度 &lt;= 10000

## Solution

```python
public static String replaceSpace(StringBuffer str) {
    for (int i = 0; i < str.length(); i++) {
        if (str.charAt(i) == ' ') {
            str.replace(i, i + 1, "%20");
        }
    }
    return new String(str);
}
```
